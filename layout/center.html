居中对齐总结

margin:auto 水平居中

只能水平居中,必须指定宽度,且对浮动或绝对定位元素无效

.child{ margin:0 auto; }

text-align:center 文字水平居中

行内元素span

块级元素p

文字内容

只能对图片,文字,按钮等行内元素水平居中, 但是ie6,ie7能对任何元素居中

.parent{ text-align:center; }

line-height 和 height 等高设置单行文字垂直居中

单行文字内容

line-height 和height 设置为等高, 单行文字垂直居中,只适用一行文字情况

.parent{ height:40px; line-height:40px; }

用padding使多行文字垂直居中

多行文字内容
多行文字

使用padding上下左右一致

缺点：高度不固定

.parent{ padding:20px; }

table表格放置居中

文字内容 行内元素
块级元素

vertical-align垂直居中,text-align:水平居中 块级元素使用display:inline-block;才能水平居中

table td{ text-align: center; vertical-align: middle; } .cell{ display:inline-block; }

display:table-cell进行居中

父元素:display:table-cell;vertical-align:middle;text-align:center; 子元素:dislay:inline-block;
text-align:center可以使inline-block的子元素水平居中；
vertical-align:middle可以使块或行内元素垂直居中
可以用于实现多行文字居中

.parent{ display:table-cell; vertical-align:middle; text-align:center; } .cell{ display:inline-block; }

多个块级水平居中

关键样式：父元素text-align:center; 子元素display: inline-block;

.parent{ text-align:center; } .child{ display:inline-block; }

使用flex布局，父级元素display:flex;justify-content:center;

.parent{ display:flex; justify-content:center; }

绝对定位居中

1. left/top,margin-left/margin-top

只适用于已经知道宽度和高度的元素

父元素:position: relative;

子元素:position:absolute;left:50%;top:50%;margin-left:-25px;margin-top:-25px;

.parent{ position: relative; } .child{ position: absolute; width:50px; height:50px; left:50%; top:50%; margin-left:-25px; margin-top:-25px; }

2. margin:auto

只适用于已经知道宽度和高度的元素,只支持ie9+,firefox,chrome

父元素:position: relative;

子元素:position:absolute;left:0;top:0;right:0;bottom:0;margin:auto;

.parent{ position: relative; } .child{ position: absolute; left:0; top:0; right:0; bottom:0; margin:auto; }

3. css3 transform

绝对定位+css3 transform:translate(-50%，-50%)

关键样式：left:50%;top:50%;-webkit-transform:translate(-50%,-50%);

适用于未知高度和宽度的居中

.parent{ position: relative; } .child{ position: absolute; left:50%; top:50%; transform:translate(-50%,-50%); }

flex布局居中

父元素：display:flex;justify-content:center;align-item:center;

或者子元素：align-self:center;

.parent{ display:flex; justify-content:center; align-item:center; }
